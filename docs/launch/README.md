## Halt boundaries (per `Decisions` policy)

The agent drafts these files but does NOT:

- Send emails (external comm).
- Post to Twitter, HN, Reddit (external comm).
- Publish the blog post (external comm).
- Submit the workshop paper (external comm + needs real-API numbers).
- Create real GitHub issues from `docs/good-first-issues.md` (external comm).
- Push to a git remote.

Each file flags its halt note inline.
